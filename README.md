# Python目标检测系统 python617

## 项目概述

本项目是一个基于YOLOv8的深度学习目标检测系统，专门用于行人检测。该系统使用Python语言开发，集成了PyTorch深度学习框架。

## 技术栈

- **核心框架**：PyTorch
- **Python版本**：不详（建议使用Python 3）
- **深度学习模型**：YOLOv8
- **前端界面**：PyQt5

## 功能模块

- **模型训练**：使用自定义数据集进行模型训练
- **模型部署**：将训练好的模型集成到前端界面
- **目标检测**：对图片进行行人检测

## 系统角色

- **训练模型**：train_model.py
- **前端界面**：qt.py
- **预训练模型**：best.pt

## 运行环境

- 硬件要求：CPU（建议使用具备GPU加速的设备）
- 软件要求：PyTorch 2.3.1，PyQt5

## 部署步骤

1. 确保Python环境及依赖项已正确安装
2. 修改`train_model.py`和`qt.py`中的相关路径
3. 运行`qt.py`启动前端界面

## 目录结构

```
project/
```

## 核心亮点

- 基于先进的YOLOv8算法，实现高效的目标检测
- 集成PyQt5构建友好的用户界面
- 提供模型训练和部署的一站式解决方案

**注意：** 由于数据集的限制，模型的识别准确率可能不高。在部署和使用过程中，需根据实际情况调整模型和代码。
